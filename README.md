# 📊 Revenue & Billing Analytics Dashboard (Power BI)

## 🔍 Overview
This project showcases a multi-page Power BI dashboard built to analyze revenue trends, billing performance, contract risks, and customer insights.

## 📊 Key Features
- KPI tracking: Total Revenue, Paid, Outstanding, Overdue
- Revenue trend analysis
- Billing analysis (Manual vs Auto, Status)
- Contract analysis (Active, Expiring, Expired)
- Customer insights (Top customers, payment behavior)

## 🛠 Tools Used
- Power BI
- DAX
- Excel

## 📈 Insights
- Identified overdue revenue impacting cash flow
- Highlighted contract expiration risks
- Analyzed customer contribution to revenue
